# BranchFingerprint Status Table

| Status | Input condition | Value shape | Validity mapping | Conditioning behavior | Protocol behavior | Refusal or indeterminate behavior |
|---|---|---|---|---|---|---|
| `slope_flow_observed` | enough usable samples, no models | `SlopeFlowComparisonValue` with samples and segment evidence | defined, finite, selectable, not advanceable, observable | well-conditioned | accepted as usable transfer evidence | no model selection |
| `slope_model_residuals` | models supplied, no declared model band | slope-flow value with model residuals | defined, finite, selectable, not advanceable, observable | well-conditioned | accepted as usable transfer evidence | residual evidence only |
| `slope_model_unique_match` | exactly one model inside declared band | slope-flow value with selected model | defined, finite, selectable, not advanceable, observable | well-conditioned | accepted as usable transfer evidence | none |
| `slope_model_ambiguous` | multiple models inside declared band | slope-flow value with residuals | defined, finite, not selectable, not advanceable, observable | warning | maps to fingerprint unidentified | no unique model |
| `slope_model_no_match` | no models inside declared band | slope-flow value with residuals | defined, finite, not selectable, not advanceable, observable | warning | maps to fingerprint unidentified | no model selected |
| `slope_flow_insufficient_samples` | fewer than two loggable samples | slope-flow value preserving raw samples | not defined, not finite, not selectable, not advanceable, observable | warning | maps to fingerprint incomplete | evidence insufficient |
| `slope_flow_indeterminate` | ProjectiveRatio slope scalarization refuses or log-domain path fails | segment evidence with refusal reason | not defined, not finite, not selectable, not advanceable, observable | warning | maps to fingerprint incomplete | no sentinel slope |
| `slope_flow_proxy_uncalibrated` | reserved proxy slope-flow refusal status | slope-flow evidence | not defined, not finite, not selectable, not advanceable, observable | warning | maps to fingerprint proxy-uncalibrated | proxy missing |
| `kq_flow_stable` | all D log Kq segments inside declared band | `KqSlopeStabilityValue` with traces and stability flag | defined, finite, selectable, not advanceable, observable | well-conditioned | accepted by Kq stable rule | none |
| `kq_flow_unstable` | at least one D log Kq segment outside declared band | Kq stability value | defined, finite, not selectable, not advanceable, observable | warning | refused by Kq stable rule | proxy fingerprint blocked |
| `kq_flow_indeterminate` | missing band, insufficient samples, or slope evidence not defined | Kq stability value | not defined, not finite, not selectable, not advanceable, observable | warning | refused by Kq stable rule | proxy fingerprint blocked |
| `kq_flow_uncalibrated` | pointwise calibration invalid, indeterminate, uncalibrated, or wrong family | Kq stability value with refusal reasons | not defined, not finite, not selectable, not advanceable, observable | warning | refused by Kq stable rule | proxy fingerprint blocked |
| `fingerprint_complete` | usable projection and transfer evidence, plus stable Kq if proxy | `BranchFingerprintValue` | defined, finite, selectable, not advanceable, observable | well-conditioned | emitted by fingerprint protocol | evidence-complete only |
| `fingerprint_unidentified` | usable projection but ambiguous or no-match model evidence | BranchFingerprint value | defined, finite, not selectable, not advanceable, observable | warning | emitted by fingerprint protocol | no unique model |
| `fingerprint_incomplete` | projection or transfer evidence incomplete | BranchFingerprint value | not defined, not finite, not selectable, not advanceable, observable | warning | emitted by fingerprint protocol | raw evidence preserved |
| `fingerprint_proxy_uncalibrated` | proxy mode without stable Kq evidence | BranchFingerprint value | not defined, not finite, not selectable, not advanceable, observable | warning | emitted by fingerprint protocol | proxy evidence missing or unstable |
